# Transportation Data Analysis (Python)

Portfolio project: cleaning + exploratory analysis on transportation delay data.

## What's inside
- `transportation_analysis.ipynb` — the notebook
- `data/transportation_synthetic.csv` — sample dataset (you can replace with a real one)
- `/assets` — add screenshots of charts

## How to run
```bash
pip install -r requirements.txt
```

Open the notebook and run all cells.
